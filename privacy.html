Política de Privacidade

Última atualização: 26 de abril de 2025

Resumo: coletamos apenas os dados necessários para o funcionamento do app. Seus dados financeiros são privados e isolados — nenhum outro usuário, e nem nós, temos acesso ao conteúdo dos seus gastos.

1. Quem somos

O Finan.RB é um serviço de controle financeiro pessoal e empresarial operado por [Weberth Bruno Gomes Sobrinho], inscrita sob o CPF 054.067.543-19, com sede em São Paulo-SP, Brasil. Contato: contato@financrb.luxea.com.br

2. Dados que coletamos

Coletamos apenas os dados necessários para o funcionamento do serviço:

  • Dados de cadastro: nome e endereço de e-mail, fornecidos por você no momento do registro
  • Dados financeiros: descrição, valor, data, categoria e tipo (PF/PJ) dos gastos que você registra
  • Dados de configuração: preferências de tema, dia de fechamento do ciclo e chave de API Anthropic (armazenada de forma criptografada)
  • Dados de pagamento: processados diretamente pelo Stripe. Não armazenamos dados de cartão de crédito
  • Dados de uso: logs de acesso e informações técnicas para diagnóstico de erros (sem rastreamento de comportamento detalhado)

3. Como usamos os dados

Os dados coletados são usados exclusivamente para:

  • Autenticar sua identidade e proteger sua conta
  • Armazenar e sincronizar seus registros financeiros entre dispositivos
  • Processar pagamentos e gerenciar sua assinatura
  • Enviar comunicações transacionais (confirmação de cadastro, recuperação de senha)
  • Melhorar o produto com base em métricas agregadas e anônimas

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins publicitários.

4. Onde os dados são armazenados

Os dados são armazenados no Supabase (supabase.com), plataforma com infraestrutura na AWS (regiões dos EUA). O Supabase utiliza Row Level Security (RLS), o que significa que seus dados são tecnicamente isolados dos dados de outros usuários — nem mesmo nós conseguimos acessar o conteúdo dos seus gastos sem comprometer o banco de dados.

Ao usar o Finan.RB, você concorda com a transferência internacional de dados para os servidores do Supabase, nos termos da LGPD (art. 33, inciso I — país com nível de proteção adequado).

5. Retenção de dados

Seus dados são mantidos enquanto sua conta estiver ativa. Ao solicitar a exclusão da conta, todos os seus dados (gastos, orçamentos, perfil) são removidos permanentemente em até 30 dias. Dados de pagamento são retidos pelo Stripe conforme as obrigações legais aplicáveis.

6. Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar seus dados pessoais
  • Corrigir dados incompletos, inexatos ou desatualizados
  • Solicitar a portabilidade ou exclusão dos seus dados
  • Revogar o consentimento a qualquer momento

Para exercer qualquer um desses direitos, entre em contato pelo e-mail privacidade@financrb.com.br. Respondemos em até 15 dias úteis.

7. Cookies e rastreamento

O Finan.RB utiliza apenas cookies essenciais para manter sua sessão autenticada. Não utilizamos cookies de rastreamento ou publicidade comportamental.

8. Segurança

Adotamos as seguintes medidas de segurança:

  • Comunicação criptografada via HTTPS/TLS em todas as requisições
  • Senhas armazenadas com hash seguro (bcrypt via Supabase Auth)
  • Isolamento de dados por usuário via Row Level Security
  • Chaves de API armazenadas de forma criptografada

9. Menores de idade

O Finan.RB não é direcionado a menores de 18 anos. Não coletamos intencionalmente dados de menores. Se você acredita que um menor registrou uma conta, entre em contato conosco para remoção imediata.

10. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos usuários ativos por e-mail sobre mudanças relevantes. O uso continuado do serviço após a notificação constitui aceite das alterações.
